
# Presença360
Um projeto com o intuito de otimizar processos necessários dentro da sala de aula. A partir de observações do nosso time, tivemos a percepção que é gasto muito tempo com a realização da lista de presença, e nosso projeto traz uma solução para esse problema.

- Cadastro de Alunos

- Login de Alunos
- Cadastro de Professor
- Login de Professor
- Armazenamento de Listas de chamadas
